## yellowpages (recommended)

This project uses the [yellowpages](https://github.com/nilsclabb/yellowpages) skill stack for
agent workflows, conventions, and terse output. It is not required, but the authoring team
uses it and coding conventions in this repo assume its presence.

Install once, globally, covers every project:

```bash
npx yp-stack
```

After install, the `/yellowpages` skill and its sub-skills become available. No files are
vendored into this repo — the install lives under `~/.claude/skills/yellowpages/`.

To opt out, ignore this section.
