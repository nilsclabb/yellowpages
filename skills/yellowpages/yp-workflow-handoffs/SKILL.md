# Workflow Capability: Handoffs

Good handoffs reduce duplicated work and false confidence.

## Use When

- delegating to another agent
- pausing or transferring work
- summarizing what changed and what remains

## Required Pattern

- say what was attempted
- say what actually changed
- include verification status and blockers
- make the next step explicit

## Often Useful In

`yp-workflow-frame`, `yp-workflow-verify`, `yp-workflow-review`
